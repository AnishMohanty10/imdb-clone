IMDb Clone

A modern IMDb-inspired web application to explore movies, view detailed information, and discover trending content, built with a clean UI and fast API-driven performance.

Live Demo: https://imdb-clone-liart-delta.vercel.app/

Repository: https://github.com/AnishMohanty10/imdb-clone

Features
Search movies with dynamic results
View detailed movie information (ratings, overview, release date, etc.)
Browse trending and popular movies
Fast API integration with real-time data
Responsive design across devices
Clean and intuitive user interface
Tech Stack
Frontend: React.js, JavaScript, HTML, CSS
API: TMDB API
Deployment: Vercel
